# KPI Ticker Widget

Este widget muestra indicadores financieros en tiempo real usando TradingView.

### 📦 Instrucciones

1. Este archivo se puede publicar con GitHub Pages.
2. Incluye el `index.html` con el widget ticker tape.
3. Puede integrarse en Webex Contact Center mediante un `iframe`.

### 🌐 Ejemplo de uso

```json
{
  "comp": "agentx-wc-iframe",
  "attributes": {
    "src": "https://<TU_USUARIO>.github.io/<TU_REPO>/",
    "height": "50px",
    "width": "100%",
    "frameborder": "0",
    "scrolling": "no"
  }
}
```
